# Supported Commands

init
  - Initialize a repository

add <file>
  - Stage a file

commit -m "message"
  - Create a commit snapshot

branch <name>
  - Create a new branch

checkout <name>
  - Switch to a branch

log
  - Show commit history

diff <commit1> <commit2>
  - Show differences between commits

merge <branch>
  - Merge branch into current branch
